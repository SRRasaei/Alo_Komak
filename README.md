# 🚗 الو کمک – سامانهٔ مدیریت خدمات امداد خودرو

![Django](https://img.shields.io/badge/Django-4.2-092E20?logo=django&style=flat-square)
![Python](https://img.shields.io/badge/Python-3.9%2B-blue?logo=python&style=flat-square)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)
![Status](https://img.shields.io/badge/Status-Active-brightgreen?style=flat-square)

**الو کمک** یک نرم‌افزار تحت وب برای مدیریت کسب‌وکارهای امداد خودرو و خدمات حضوری است. این سامانه با ایجاد یک بانک اطلاعاتی کامل از مشتریان، خودروها و تاریخچهٔ خدمات، به اپراتورها امکان می‌دهد در سریع‌ترین زمان ممکن مشتری را شناسایی کرده، درخواست خدمت ثبت کنند و وضعیت آن را پیگیری نمایند.

با استفاده از **الو کمک** می‌توانید:
- مشتریان و خودروهای آنان را با جزئیات کامل (پلاک استاندارد چهاربخشی ایران) ثبت کنید.
- با جست‌وجوی سریع بر اساس شماره موبایل، نام یا پلاک، در کمتر از یک ثانیه به پروفایل مشتری و تاریخچهٔ خدمات دسترسی داشته باشید.
- درخواست‌های خدمت (امدادی/حضوری) را به‌صورت مستقیم از صفحهٔ اپراتور ثبت کنید.
- وضعیت هر درخواست را به‌راحتی تغییر دهید (در انتظار، در حال انجام، پایان‌یافته و ...).
- از پنل مدیریت جنگو برای مدیریت کامل موجودیت‌ها و تهیهٔ گزارش‌های اولیه استفاده کنید.

## ویژگی‌های کلیدی

- **مدیریت مشتریان**: ثبت اطلاعات کامل مشتری (نام، شماره تماس، کد ملی و ...) به همراه جست‌وجوی پیشرفته.
- **ثبت خودرو با پلاک استاندارد ایران**: پلاک به چهار بخش (دو عدد، حرف، سه عدد، کد شهر) تفکیک می‌شود و نمایش آن در محیط راست‌به‌چپ فارسی کاملاً صحیح است (LTR).
- **تاریخچهٔ کامل خدمات**: تمامی درخواست‌های قبلی هر مشتری با جزئیات نوع خدمت، هزینه و وضعیت قابل مشاهده است.
- **جست‌وجوی سریع اپراتور**: یک صفحهٔ ساده و کارآمد برای اپراتور که با وارد کردن شماره موبایل، نام یا بخشی از پلاک، بلافاصله پروفایل مشتری و آخرین خدمات را نشان می‌دهد.
- **ثبت سریع درخواست خدمت**: دکمهٔ "ثبت خدمت جدید" در کنار هر نتیجه جست‌وجو؛ اپراتور بدون ترک صفحه می‌تواند خدمت جدیدی برای مشتری ثبت کند.
- **مدیریت وضعیت درخواست‌ها**: تغییر وضعیت هر خدمت با یک منوی کشویی در همان صفحهٔ جست‌وجو (بدون رفرش دستی).
- **پنل مدیریت**: استفاده از Django Admin برای مدیریت شعب، کارمندان، انواع خدمت و گزارش‌گیری اولیه.
- **آماده برای گسترش**: معماری ماژولار جنگو امکان افزودن امکاناتی مانند ارسال پیامک یادآوری، اپ موبایل، گزارش‌های تحلیلی و ... را به‌راحتی فراهم می‌کند.

## فناوری‌های استفاده‌شده

| حوزه | فناوری |
|------|--------|
| Backend | Django 4.2 (Python) |
| Database | SQLite (توسعه) / PostgreSQL (توصیه برای محیط عملیاتی) |
| Frontend | HTML5, CSS3, Django Templates (با قابلیت جایگزینی با React در آینده) |
| ابزارهای جانبی | python-dotenv (مدیریت متغیرهای محیطی) |
| محیط توسعه | Visual Studio 2026 / VS Code |

## پیش‌نیازها

برای اجرای پروژه روی سیستم خود، نیاز دارید:

- **Python** نسخهٔ 3.8 یا بالاتر
- **pip** (مدیر بستهٔ پایتون)
- **Git** (برای کلون کردن مخزن)
